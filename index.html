Sourcestats by zkarpinski

A web statistics application based in rails for VALVe game servers (TF2, L4D, CS:S etc...)

Dependencies

mislav-will_paginate

Install

# Install will paginate rake gems:install # If using MySQL create your database, substituting localhost for the hostname or IP address of your database. Be sure to pick a different password and/or username mysqladmin create database sourcestats_production mysql -u root -p -h localhost mysql grant all on sourcestats_production.* to pyro@localhost identified by 'PASSWORD' flush privileges; # Run database migrations rake db:migrate # Initialize the log listen script cd /web/sites/sourcestats.com ruby script/runner -e development SourcestatsLogger.listen # Configure your source game server to turn on logs and send them to our logger. Editing your server.cfg: log on log_addaddress localhost:20808 # Join your server and see if it works!
